# Multiple uploads

Project made to learn about upload files with Node.js and ReactJS.
It was made following the Rocketseat tutorial, (click here)[https://www.youtube.com/watch?v=G5UZmvkLWSQ] to access.

## Stack

<ul>
  <li>Backend</li>
    <ul>
      <li>Node.js</li>
      <li>Express</li>
      <li>MongoDB</li>
    </ul>
  <li>Frontend</li>
    <ul>
      <li>ReactJS</li>
      <li>Axios</li>
      <li>Styled components</li>
    </ul>
</ul>
